# 考勤管理系统 (Attendance Management System)

纯前端单页应用（SPA），无需后端服务，所有数据存储在浏览器 IndexedDB 中。

## 功能特性

### 数据导入
支持 5 种 Excel 文件导入，自动识别格式：
- **打卡机数据（精简版）** - 原始打卡记录，含部门、姓名、日期时间等
- **打卡机数据（详细版）** - 处理后的考勤汇总，含签到签退、迟到早退等
- **OA 请假申请单** - 调休、公休、丧假等各类请假记录
- **OA 加班申请单** - 加班起止时间、加班内容
- **OA 出差申请单** - 出差人员、目的地、起止日期

### 数据匹配与合并
- 按 **姓名** 自动匹配所有数据源
- 每人每天生成统一考勤记录，关联打卡、请假、加班、出差信息
- 自动检测异常：漏打卡、迟到、早退、旷工

### 数据展示
- **考勤明细** - 可按部门、人员、月份、状态筛选的表格视图
- **日历视图** - 按月展示，颜色标识出勤状态（绿色正常/黄色异常/红色缺勤）
- 支持节假日与休息日管理

### 模板化导出
- 上传 Excel 模板，使用 `{{占位符}}` 语法（如 `{{姓名}}`、`{{签到时间}}`）
- 系统自动识别并填充数据
- 支持多模板管理与选择导出

## 技术栈

| 技术 | 用途 |
|------|------|
| [Dexie.js](https://dexie.org/) | IndexedDB 封装，本地数据存储 |
| [SheetJS (xlsx)](https://sheetjs.com/) | Excel 文件导入与导出 |
| [TailwindCSS](https://tailwindcss.com/) | UI 样式框架 |

## 使用方式

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。

默认账号密码：`admin` / `admin123`

## 页面说明

| 页面 | 功能 |
|------|------|
| 仪表盘 | 总览统计：员工数、出勤率、异常数 |
| 数据导入 | 拖拽上传 Excel 文件，查看导入历史 |
| 考勤明细 | 筛选、查看所有考勤记录明细 |
| 日历视图 | 按月日历展示，点击查看当日详情 |
| 导出中心 | 模板管理、数据导出 |
| 考勤设置 | 工作时间、打卡规则配置 |
